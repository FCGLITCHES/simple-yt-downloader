How to get cookies:

  1. Install a browser extension like cookies.txt for Chrome/Firefox
  2. Go to the website where you have permission to access content
  3. Click the extension and export cookies for the relevant domain
  4. Save the file and upload it here, or copy-paste the content
  5. Click Save to apply the cookies
