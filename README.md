## ğŸ› Troubleshooting

### Downloads Not Starting

- Check your internet connection
- Verify the URL is correct
- Try updating tools via Settings
- Restart the application

### Low Quality Downloads

- Make sure you're selecting the highest available quality
- Import YouTube cookies for better quality access
- Check if the video has higher quality options available

### Playlist Downloads Failing

- Try reducing concurrent downloads (set to 1)
- Check your internet connection stability
- Reupload cookies file
- Some videos in playlists may be unavailable

### FFmpeg Errors

- Go to Settings â†’ Update Tools
- Check if FFmpeg is working
- Re-download FFmpeg if needed from [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html)
